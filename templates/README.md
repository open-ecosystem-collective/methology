# Templates

**Open Ecosystem Collective – Output Templates**

These templates are the deliverables clients receive at the end of each engagement.
They are starting points – adapt them to the specific client and situation.

| Template | For | Engagement |
|---|---|---|

| [`collaboration-plan-template.md`](collaboration-plan-template.md) | Level 3 Collaboration Plan | 3–4 days · CHF 5'500 |

---

## How to use these templates

1. Fill in the client-specific details (replace all `[placeholders]`)
2. Remove sections that are not relevant to this specific client
3. Add context that makes the report more useful for this client
4. Deliver as PDF or shared document – not as raw Markdown

The templates are written in Markdown for easy editing and version control.
Convert to PDF before sharing with clients.
